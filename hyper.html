HTML stands for Hypertext Markup Language. HTML is the base for any website.

How can I code a website?

The first thing you can do is find an IDE. This is how you will code a website. One of my favourite online ones is Repl.it. Once you know where you will code move onto the next step.
Next you need to start a new file. If you are on repl, this is easy, just press the blue plus sign.
The first thing you should do in your file is start typing some text I would start with a title try typing this: <h1>My Website</h1>. Then at the top you should see a tag that says <title> change the text inside from repl.it to whatever you want.
